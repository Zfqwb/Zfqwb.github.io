Object类

概述

​ Object是类层次结构的根类, 是一切类的祖宗.
​ 所有的类都直接或者间接的继承自Object类.

方法

​ public String toString();
​ 返回对象的字符串表示形式
​ 子类重写后,控制台直接打印会比较方便
​ public boolean equals(对象);
​ 比较两个对象是否相等.默认比地址值
​ 子类重写后,重写后比内容.

BigDecimal

概述

​ 是一个比double类型更大的小数
​ 该类的精度比double更加的精确

构造方法

​ public BigDecimal(double d);
​ public BigDecimal(String d);
​ //推荐使用,因为更加精确

基本运算

​ public BigDecimal add(另一个BigDecimal对象)
​ //加法
​ public BigDecimal subtract(另一个BigDecimal对象)
​ //减法
​ public BigDecimal multiply(另一个BigDecimal对象)
​ //乘法

包装类

概述

​ 就是基本数据类型对应的”引用数据类型形式”

作用

​ 基本数据类型和String之间的相互转换

组成

​

基本数据类型 包装类
byte Byte
short Short
int Integer
long Long
float Float
double Double
char Character
boolean Boolean

概念

装箱

​ 把”基本数据类型”变为”与之对应的包装类”
​ 自动装箱: 指的就是自动发生的装箱操作
​ JDK5

拆箱

​ 把”包装类”变为”与之对应的基本数据类型”
​ 自动拆箱: 指的就是自动发生的拆箱现象
​ JDK5

Arrays

概述

​ 专门用来操作数组的工具类.

方法:

​ public static void sort(int[] arr);
​ //对数组进行升序排列
​ public static int binarySearch(int[] arr, int key);
​ //二分查找数组arr中的key元素所在的索引位置.
​ //二分查找时,要求数组必须是有序的, 而且是从小到大的顺序
​ public static String toString(int[] arr);
​ //返回数组的字符串形式.(“[1,2,3]”)
