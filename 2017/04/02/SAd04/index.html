内部类&Lambda

内部类

概述

​ 方在类的内部的类,就称之为”内部类”

分类

成员内部类

​ 类中方法外的内部类

局部内部类

​ 方法中的内部类

成员内部类(了解)

位置

​ 类中方法外

语法

创建对象

​ 外部类名.内部类名 对象名 = new 外部类名().new 内部类名();

使用对象

​ 对象名.成员变量
​ 对象名.成员方法

Lambda

概述

​ 可以理解为”是对匿名内部类的再次简化”

要求

​ 父级必须是接口
​ 接口中有且仅有一个抽象方法

格式

​ ( ) -> { }
​ ( ) : “重写的方法”的形参
​ { } : “重写的方法”的方法体
