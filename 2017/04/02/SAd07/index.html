Collection集合

集合体系

Collection

概述

​ 单列集合顶层接口
​ 就是集合中一次只能存取一个元素

分支

List
特点

​ 有序(存储和遍历的顺序一致)
​ 有索引
​ 元素可以重复

分支
ArrayList

​ 底层结构是数组,查询快,增删慢
​ 线程不安全,效率高.

LinkedList

​ 底层数据结构是链表, 查询慢,增删快
​ 线程不安全,效率高.

Set
特点

​ 无序(存储和遍历的顺序不一致)
​ 无索引
​ 元素不可以重复

分支
HashSet
LinkedHashSet
TreeSet

Map

概述

​ 双列集合顶层接口
​ 就是集合中一次存取一对元素
​ 元素必须成对出现

分支

HashMap
TreeMap
Hashtable

int size() 集合的长度,也就是集合中元素的个数

​ E next():
​ 从集合中获取下一个元素

​ 删除正在遍历的当前元素
