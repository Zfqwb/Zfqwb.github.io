Map集合

HashSet

概述

​ HashSet是Set集合的子类
​ HashSet也是”无序,无索引,元素不可以重复”

原理(元素不重复原理)

​ HashSet会自动对比元素的equals方法和hashCode方法
​ 如果一样就不存储
​ HashSet的底层是Hash表

操作

​ HashSet中存储的元素,必须重写hashCode和equals方法

扩展

LinkedHashSet

​ 是HashSet的子类
​ Set体系里边唯一有序的set集合

Collections

单列集合的工具类
里边包含了很多操作集合的方法
常用方法
方法 说明
public static void sort(List list) 对list集合进行排序,顺序是自然顺序
public static void sort(List list,Comper c) 按照指定排序规则,对list集合进行排序
public static void reverse(List<?> list) 让list集合中的元素反转
public static void shuffle(List<?> list) 随机打乱list集合中元素的顺序

原理

​ 默认创建一个长度为16, 加载因子0.75的数组
​ 当数组中存储的元素超过16*0.75时
​ 数组的长度扩容为原来的2倍
​ 数组中的每个位置都维护一个链表
​ 链表的长度超过8时, 该链表会自动转换为红黑树

Map集合

概述

​ 是一个双列集合.也叫”键值对”集合.

特点:

​ 1.”键”和”值”必须成对出现.
​ 2.”键”不允许重复,”值”可以重复.
​ 3.”键”和”值”结合在一块,称之为一个”键值对”, 是集合的一个元素.

方法

​

方法 说明
V put( K key,V val) 添加元素,如果键不存在则添加,如果键存在则覆盖
V remove(K key) 根据键,删除键值对元素
void clear() 清空集合
boolean containsKey(K key) 判断集合中是否包含指定的键
boolean containsValue(V val) 判断集合中是否包含指定的值
boolean isEmpty() 判断集合是否为空
int size() 获取集合的长度
V get(Object key) 根据键获取值
SetkeySet() 获取所有键的集合
Collectionvalues() 获取所有值的集合
Set<Map.Entry<K,V>> entrySet() 获取所有键值对对象的集合

可变参数

概念:

​ 用于方法的形参上,当参数类型确定,参数个数不确定的时候使用.

格式:

​ 修饰符 返回值类型 方法名(数据类型… 变量名) { }

注意事项:

1.可变参数本质上是一个数组.

2.一个方法中最多只有一个可变参数

3.可变参数必须放在方法参数的最后

数组转换为集合

JDK9之前:

​ List list = Arrays.asList(T… t);
​ //把数组转换为集合.该集合不允许添加和删除,但可以修改

JDK9之后:

List list = List.of(T… t);

​ //把数组转换为集合.该集合不允许添加,删除和修改

Set set = Set.of(T… t);

​ //把数组转换为集合.该集合不允许添加,删除和修改

Map<K,V> map = Map.ofEntries(Map.Entry<K,V>… entries);

​ //把数组转换为集合.该集合不允许添加,删除和修改

注意:

​ 数组转换的集合,长度都不允许发生变化.
​ 如果想让集合再次发生变化,可以利用集合的构造方法,重新创建新的集合.
​ ArrayList arrayList = new ArrayList<>(数组转换后的集合);
