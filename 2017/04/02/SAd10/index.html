Stream&File

Stream流

概述:

​ 专门用来快速的处理集合或数组的一个工具.

获取:

​

数据类型 如何转换为Stream流
Collection集合 Stream s=集合.stream();
Map集合->键 Stream s=集合.keySet().stream();
Map集合->值 Stream s=集合.values().stream();
Map集合->键值对 Stream s=集合.entrySet().stream();
数组 Stream s=Stream.of(数组名)
多个零散元素 Stream s=Stream.of(数组元素1,数组元素2…)

test

操作:

​

方法名 说明
public Stream filter(Predicate predicate) 按照指定的Predicate规则,对流中数据进行过滤
public Streamlimit(long maxSize) 获取前几个元素
public Streamskip(long n) 跳过指定参数个元素
public static Stream concat(Stream a,Stream b) 合并a和b两个流为一个流
public Stream distinct() 移除流中重复的元素
public void forEach( Consumer action ) 按照指定Consumer规则,遍历流中数据
public long count() 返回此流中的元素数
public Stream map(Function mapper) 把当前Stream流转换为其他类型的Stream流

路径问题

绝对路径:

​ 是绝对不会发生变化的路径.
​ 一般是从盘符开始.

相对路径:

​ 是根据某个固定参照物,对比出来的路径.
​ 在Java程序中,默认参照物是”当前项目”

注意:

​ 当前所说的路径, 指的是Java项目的路径
​ Web项目中的相对和绝对路径会有少许差别
