接口&多态

代码块

概述

​ 被一对大括号包围的一块代码

分类

局部代码块

位置

​ 方法中

作用

​ 限制变量的使用范围

test

构造代码块

位置

​ 类中方法外

作用

​ 抽取构造方法的重复代码

注意

​ 只能抽取构造方法中开头的相同代码
​ 构造代码块是在构造方法之前执行

test

静态代码块

位置

​ 类中方法外

语法

​ static{ … }

特点

​ 在类加载时静态代码块执行
​ 静态代码块只能执行一次

场景

​ 用户加载配置文件
​ 加载驱动

接口

概述

​ 类中的方法全是抽象方法时, 该类就可以设置为”接口”

语法

定义

​ public interface 接口名{ }

实现

​ public class 类名 implements 接口名{ }

特点

​ 接口中的方法全是抽象方法
​ 子类必须重写所有的抽象方法

成员

成员变量

​ 只能是静态常量
​ 默认修饰符 public static final

成员方法

​ 只能是抽象方法
​ 默认修饰符 public abstract

构造方法

​ 接口没有构造方法

接口新特性

默认方法

格式

​ public default 返回值类型 方法名(参数…){ }

注意

​ 子类可以重写, 也可以不重写, 不做强制要求
​ 如果重写, 在重写方法中不能携带default关键字
​ 如果多个接口中出现相同的默认方法, 则子类必须重写
​ JKD8之后才能使用

静态方法

格式

​ public static 返回值类型 方法名(参数…){ }

注意

​ 只能通过接口名调用->”接口名.方法()”
​ JKD8之后才能使用

私有方法

格式

​ private 返回值类型 方法名(参数…){ }
​ private static 返回值类型 方法名(参数…){ }

应用

​ 一般用于抽取默认方法,或者静态方法重复部分
​ JKD9之后才能使用
