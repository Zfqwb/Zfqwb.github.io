JavaSE进阶-Day04 内部类&Lambda

本文最后更新于：1 小时前

内部类&Lambda

内部类
概述
方在类的内部的类,就称之为”内部类”
分类
成员内部类
类中方法外的内部类
局部内部类
方法中的内部类

成员内部类(了解)
位置
类中方法外
语法
创建对象
外部类名.内部类名 对象名 = new 外部类名().new 内部类名();
使用对象
对象名.成员变量
对象名.成员方法
分类
静态内部类
被static修饰的成员内部类
外部类.内部类 对象名 = new 外部类.内部类();
私有内部类
被private修饰的成员内部类
只能在他所在的外部类中,创建对象并使用.

局部内部类
位置
方法内部
注意
只能在方法内部进行使用
局部内部类使用完毕之后, 会被立即销毁
场景
临时性的创建子类对象
用完之后立即销毁

匿名内部类
概述
没有名字的局部内部类
是局部内部类的简化写法
格式

注意
    匿名内部类本质上是"一个继承父类或实现接口的子类的对象"

Lambda
概述
可以理解为”是对匿名内部类的再次简化”
要求
父级必须是接口
接口中有且仅有一个抽象方法
格式
( ) -> { }
( ) : “重写的方法”的形参
{ } : “重写的方法”的方法体
省略规则
参数的数据类型可以省略
如果参数只有一个,则()可以省略
如果方法体只有一句话,则可以同时省略”{},分号和return”

Math
概述
包含了很多数学运算的工具类.
方法
public static int abs(int num);
//求绝对值
public static double ceil(double num);
//向上取整(取大于等于num的最小整数的double类型形式)
public static double floor(double num);
//向下取整(取小于等于num的最大整数的double类型形式)
public static int max(int a,int b);
//求两个数的最大值
public static double round(double num);
//四舍五入
public static double pow(double a, double b);
//计算a的b次幂
public static double random();
//产生0.0-1.0之间的随机数,包含0.0,不包含1.0

System
概述

方法

//获取当前系统时间所代表的毫秒值(从1970.01.01.00:00:00开始计算)
