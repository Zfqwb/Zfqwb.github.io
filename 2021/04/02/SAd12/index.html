对象流

概述

​ 专门用来读写对象的IO流

分类

对象输出流(序列化流):

类名

​ ObjectOutputStream

构造方法

​ public ObjectOutputStream(OutputStream os);

特有方法:

​ void writeObject(Object obj);

对象输入流(反序列化流):

类名

​ ObjectInputStream

构造方法

​ public ObjectInputStream(InputStream is);

特有方法:

​ Object readObject();

注意

​ 1.被序列化的对象必须实现Serializable接口
​ 2.被序列化的类的内部需要添加一个序列化号
​ private static final long serialVersionUID = 1L;
​ 在类的后边选择”Alt + 回车”
​ 3.如果某个成员变量不想参与序列化,则可以使用transient关键字修饰
​ private transient int age;
​ 4.序列化对象的个数 == 反序列化对象个数

Properties类

概述:

​ Properties本质上是一个Map集合.
​ 专门用于读取配置文件(文件中只包含键值对)

特有方法:

特有方法 说明
void setProperty(String key,String value); 就相当于map集合的put方法
String getProperty(String key); 就相当于map集合的get方法
void load(Reader fr); 把配置文件中的键值对,读取到Properties集合中.
void store(Writer fw, String comments); 把Properties集合中的键值对,同步到配置文件中.
