Hexo安装与自定义主题

本文最后更新于：5 天前

安装 Hexo 相当简单，只需要先安装下列应用程序即可：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • Git

安装node.js

node.js官网

输入 “node -v” 来测试是否成功安装

输入 “npm -v” 来测试是否成功安装

安装git

git官网

输入 “git –version” 来测试是否成功安装

安装Hexo

npm是从国外服务器下载，速度慢可能出现异常，所以我们可以安装cnpm来替换npm
